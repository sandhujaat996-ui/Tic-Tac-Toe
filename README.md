🎮 Tic Tac Toe Game

📌 Description

This is a simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. It allows two players to play the game in a browser with a clean and user-friendly interface.

🚀 Features

- Two-player mode
- Real-time win detection
- Draw game detection
- Game reset functionality
- Responsive design

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

🎯 How to Play

1. The game is played between two players (X and O).
2. Players take turns to mark a box.
3. The first player to align three marks (row, column, or diagonal) wins.
4. If all boxes are filled and no one wins, the game ends in a draw.
